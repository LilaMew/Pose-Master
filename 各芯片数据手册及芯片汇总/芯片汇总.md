# 芯片汇总
* 主控与无线信号控制 --> nRF52840
* 姿态数据采集 --> mpu-9250

* 电源管理 --> BQ24072
* 串口&RS485信号转换 --> MAX3485
* EEPROM存储器 --> AT24C512C

    ## 芯片功能介绍：
    1. nRF52840：主控芯片，负责处理传感器数据、控制无线信号等任务。
    2. mpu-9250：姿态数据采集芯片，负责采集设备的姿态数据。
    3. BQ24072：电源管理芯片，负责为设备提供稳定的电源。
    4. MAX3485：串口&RS485信号转换芯片，负责将串口信号转换为RS485信号。
    5. AT24C512C：EEPROM存储器芯片，负责存储设备的配置信息以及采集到的数据。

    ## 芯片选型原因：
    1. nRF52840：选择支持BLE和Wi-Fi功能的芯片，以满足***无线信号控制***的需求。
    2. mpu-9250：选择支持陀螺仪、加速度计和磁力计的芯片，以满足***完整姿态数据***采集的需求。
    3. BQ24072：选择支持USB充电和电池管理的芯片，以满足***电源管理***的需求。
    4. MAX3485：选择支持RS485通信的芯片，以满足***中长距离有线信号控制***的需求。
    5. AT24C512C：选择支持大容量存储的芯片，以满足存储设备配置信息和采集到的数据的需求。
